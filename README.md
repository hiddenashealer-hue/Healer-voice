# 🎤 Healer Voice

A personal learning project developed to explore speech recognition, voice recording, and language translation using Python and Flask.

## 📌 Project Status

✅ Learning project completed in 2025.

This project is currently not under active development while I focus on WordPress, WooCommerce, and AI-powered web applications.

## ✨ Features

- Voice recording
- Speech-to-text conversion
- Language translation
- Flask-based web application

> **Note:** During testing, voice recording and speech-to-text features worked in many cases. Some functionality remains experimental and would require additional development.

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Google Translate API
- Speech Recognition

## 🎯 Purpose

This project was created to learn how speech recognition and language translation can be integrated into a Python web application.
